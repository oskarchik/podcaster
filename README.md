# Podcaster

A list of most listened podcast of iTunes in last 24h

## Quick-start and usage

1 Clone this repo

```
git clone git@github.com:oskarchik/react-typescript-boilerplate.git
```

2 Install dependencies

```
npm install
```

To run in development mode

```
npm run dev
```

and open

`http://localhost:5173/`

in your browser
